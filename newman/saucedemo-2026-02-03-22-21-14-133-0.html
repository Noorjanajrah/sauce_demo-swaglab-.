## Overview This endpoint updates an existing cart by merging new products or updating quantities of existing products. The merge operation allows you to add products to the cart without replacing the entire cart contents. ## Endpoint ``` PUT https://dummyjson.com//carts/1 ``` ## Request Body Parameters ### `merge` (boolean) - **Required**: Yes - **Description**: Controls the merge behavior for the cart update - **Values**: - `true`: Merges the provided products with existing cart items. If a product already exists in the cart, its quantity will be updated. New products will be added. - `false`: Replaces the entire cart contents with the provided products ### `products` (array) - **Required**: Yes - **Description**: Array of product objects to add or update in the cart - **Structure**: Each product object contains: - `id` (number): The unique identifier of the product - `quantity` (number): The desired quantity for this product **Example Request Body:** ``` json { "merge": true, "products": [ { "id": 1, "quantity": 10 } ] } ``` ## Response Structure The API returns the updated cart object with the following structure: ``` json { "id": 1, "products": [ { "id": 168, "title": "Product Name", "price": 32999.99, "quantity": 3, "total": 98999.97, "discountPercentage": 13.39, "discountedPrice": 85744, "thumbnail": "https://..." } ], "total": 103874.75, "discountedTotal": 89775, "userId": 33, "totalProducts": 5, "totalQuantity": 25 } ``` ### Response Fields: - **id**: Cart identifier - **products**: Array of all products currently in the cart with full details - **total**: Sum of all product totals before discounts - **discountedTotal**: Total after applying all discounts - **userId**: ID of the user who owns this cart - **totalProducts**: Count of unique products in the cart - **totalQuantity**: Sum of quantities across all products ## Merge Behavior Notes 1. **Updating Existing Products**: When `merge: true` and a product with the specified ID already exists in the cart, only the quantity is updated.
